Установка среды разработки под Ангуляр 2
=========================================

Для начала установить последнюю версию Ноды.

Затем следует установить командную строку Ангуляра:

```bash
npm i @angular/cli -g
```

Далее создаем папку под проект и в ней запускаем команду:

```bash
ng init --style=sass
```
